Text Elements & Links

The Anatomy of Content

Text elements are the core of any web page. They provide hierarchy, meaning (semantics), and connectivity. For ADHD learners, focusing on the purpose of the tag helps it stick.

Micro-Win Strategy: Don't try to memorize all tags at once. Master the "Big Three" (Headings, Paragraphs, and Links) before moving to styling tags.

Heading Elements (h1-h6)

Headings create a roadmap for the user and search engines. Use only one <h1> per page.

TagUsageVisual Hierarchy
<h1>Main Page TitleLargest / Most Important
<h2>Major SectionsLarge
<h3> - <h6>Sub-sectionsDecreasing size / Importance

The Paragraph Element

TagDefinitionExample
<p> Defines a block of text. Browsers automatically add a margin before and after. <p>This is a paragraph.</p>

The Anchor (Link) Element

Links are the "connective tissue" of the web. The href attribute is mandatory.

AttributePurposeExample
hrefThe destination URL.<a href="https://google.com">
target="_blank"Opens the link in a new tab.<a href="..." target="_blank">

Stylistic Text Elements

These tags change how text looks without necessarily adding "meaning" to the content.

TagFunctionVisual Result
<b>Bold text for attention.Bolded
<i>Italic text for technical terms/thoughts.Italicized
<u>Underlined text (use sparingly).Underlined

Semantic Formatting Elements

These tags tell the browser and screen readers that the text has specific importance.

TagMeaningUsage
<strong>High ImportanceRenders as Bold
<em>Emphasized StressRenders as Italics
<small>Side-commentsSmaller font size
<mark>Highlighted TextYellow background
<s>Inaccurate/IrrelevantStrikethrough
<del> / <ins>Deleted / InsertedOld New
